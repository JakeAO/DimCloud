# Brainstorm: Core Loops — Successor Concepts

*Compiled from comparative analysis; all content is theorycraft/non-canon.*

## Loop Definitions

### DC1: Linear Georama Loop
```
DUNGEON (Procedural)
    │
    ├─ Kill Enemies → ABS → Weapon Level
    ├─ Find Atla (Toan Only) → Inventory
    ├─ Defeat Key Enemy → Floor Key → Next Floor
    └─ Exit Dungeon
         │
         ▼
GEORAMA (Village Build)
    │
    ├─ Assemble Atla Pieces → Buildings/Villagers/Nature
    ├─ Place on Grid → Rotation/Adjacency
    ├─ Talk to Villagers → Requests (Logic Puzzles)
    ├─ Fulfill Requests → 100% Request Reward
    └─ 100% Collection + Reconstruction → Next Dungeon Unlock
         │
         ▼
VILLAGE HUB
    │
    ├─ Save / Shop / Weapon Synthesize / Character Swap
    └─ Enter Next Dungeon
         │
         ▼
    (REPEAT × 6 Chapters)
```

**Loop Time**: ~3-6 Hours per Chapter  
**Agency**: Placement Only (Pieces Pre-Determined)  
**Causality**: Dungeon → Village (One Way)

---

### DC2: Causal Time-Travel Loop
```
DUNGEON (Past Era)
    │
    ├─ Kill Enemies → ABS/SP → Weapon Level
    ├─ Defeat Key Enemy → Floor Key → Next Floor
    ├─ Collect Geostones → Carpenterion
    ├─ Photograph Objects → Photos (30 Album)
    ├─ Complete Medal Objectives → Medals
    ├─ Fish / Spheda → Side Rewards
    └─ Exit / Save
         │
         ▼
CARPENTERION → BLUEPRINTS (From Geostones)
         │
         ▼
GEORAMA (Past Build)
    │
    ├─ Buy/Find/Invent Materials
    ├─ Place Blueprints → Cost Polyn + Materials
    ├─ Earn Culture Points (CP)
    ├─ Meet 10 Conditions (Pop/CP/Specifics)
    └─ 100% Conditions → FUTURE ALTERED
         │
         ▼
TIME TRAVEL (Atlamillia)
         │
         ▼
FUTURE ERA (Monica's Time)
    │
    ├─ See Past Changes Manifested
    ├─ New Dungeon Paths / Enemy Tables / Chests
    ├─ Collect Future Chests → +HP / +Def (Guaranteed)
    ├─ New Geostones for Next Past Area
    └─ Story Advances
         │
         ▼
    (REPEAT × 7 Chapters + Post-Game)
```

**Loop Time**: ~4-8 Hours per Chapter (More Systems)  
**Agency**: Blueprint Choice + Material Gathering + Placement Logic  
**Causality**: Past → Future → Next Past (Bidirectional)

---

### Successor: Multi-Era Causal Loop
```
MULTI-ERA GEORAMA (4 Layers)
    │
    ├─ ANCIENT PAST → RECENT PAST → PRESENT → FUTURE
    │     │              │            │         │
    │     ▼              ▼            ▼         ▼
    │  Ruins         Foundations    Town      Metropolis
    │  Artifacts     Infrastructure  Economy   Tech
    │
    ▼
PLAYER ACTIONS (Any Layer)
    │
    ├─ BUILD (Parametric Blueprints) → Resource Nodes Generated
    ├─ EXCAVATE (Dungeon Seeds) → Loot Tables Defined
    ├─ CRAFT (Essence System) → Weapon/Tool Evolution
    ├─ RECRUIT (NPCs) → Passive Buffs + Quest Lines
    ├─ DIPLOMATIZE (Monster Forms) → Faction Reputation
    └─ SEED (Share Blueprints/Seeds) → Community Content
         │
         ▼
CAUSAL SIMULATION (Real-Time)
    │
    ├─ Resource Flow: Past Mines → Present Factories → Future Tech
    ├─ Population: Housing → Workers → Specialists → Leaders
    ├─ Defense: Walls → Guards → Mechs → Orbital
    ├─ Culture: Shrines → Festivals → Tourism → Soft Power
    └─ Threat: Monsters → Raiders → Warlords → Cosmic
         │
         ▼
WORLD STATE VISUALLY UPDATES (All Layers Simultaneously)
    │
    ├─ New Dungeon Seeds Generated from World State
    ├─ Blueprint Options Expand/Contract
    ├─ NPC Dialogues/Quests Change
    ├─ Economy Prices Fluctuate
    └─ Cosmetic/Visual Evolution
         │
         ▼
DUNGEON RUN (Procedural Seed + Handcrafted Anchors)
    │
    ├─ Weapon Legacy System (Component + Essence + Memory)
    ├─ Companion Synergy (Mech/Monster/Drone + Support NPCs)
    ├─ Stance Combat (Melee/Ranged/Magic/Companion)
    ├─ Per-Floor Mastery Objectives (Speed/Wipe/Photo/Fish/Build)
    └─ Loot → Essences / Blueprints / Seeds / Components
         │
         ▼
RETURN TO GEORAMA (Any Layer) — Loop Continues
```

**Loop Time**: Variable (15 Min Dungeon → Hours Georama)  
**Agency**: Full Parametric Design + Era Selection + Community Sharing  
**Causality**: Multi-Layer Simultaneous Simulation

## Loop Component Matrix

| Component | DC1 | DC2 | Successor |
|-----------|-----|-----|-----------|
| **Dungeon Input** | Atla (Fixed Pieces) | Geostones + Photos + Medals | Seeds + Essences + Blueprints + Components |
| **Dungeon Output** | Weapon ABS | ABS + SP + Geostones + Photos + Medals + Fish | Essences + Components + Seeds + Data |
| **Build Input** | Atla Pieces (Pre-made) | Blueprints + Materials + Polyn | Parametric Blueprints + Resources + Essences |
| **Build Process** | Place → Rotate → Request Check | Place → CP → Conditions → Future Change | Design → Simulate → Build → Multi-Era Ripple |
| **Build Output** | Villager Rewards (Items) | Future Changes + Stat Chests + Medals | Persistent World State + Passive Income + Unlocks |
| **Causality** | Linear (Dungeon→Build) | Bidirectional (Past↔Future) | **Omnidirectional (4 Layers + Community)** |
| **Permanent Progression** | Weapon Stats + Char Stats (Finite) | Weapon Stats + Future Chests (Guaranteed) | **Legacy Traits + World State + Community** |
| **Replay Hook** | Layout Optimization | Medals / Inventions / Photos / Badges | **Seeds / Blueprints / Legacy / Leaderboards** |
| **Time Pressure** | Thirst Meter | None (Medals Optional) | **Stamina (Active) + Threat Events (Periodic)** |

## Design Principle: "Loop Density"

**DC1**: 3 Systems in Loop (Dungeon, Weapon, Georama)  
**DC2**: 12+ Systems Interlocked  
**Successor**: 20+ Systems + Community Layer

> **Rule**: Every added system must connect to ≥3 existing systems.  
> **Rule**: No system exists in isolation — even cosmetics feed economy/reputation.
