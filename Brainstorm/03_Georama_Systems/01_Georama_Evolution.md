## DC1: Georama 1.0 — Placement Puzzle

### Pipeline
```
Dungeon → Atla Spheres (Random Pieces) → Auto-Assemble → Place on Grid
```

### Mechanics
| Aspect | Implementation |
|--------|----------------|
| **Input** | Atla Spheres (Toan Only) → Pre-made Pieces |
| **Placement** | Tile Grid; Rotation 90°; Collision Check |
| **Constraints** | Valid Terrain; Villager Requests (Logic Puzzles) |
| **Completion** | 3 Tracks: Collection% / Rebuild% / Request% |
| **Progression Gate** | Collection 100% + Rebuild 100% → Next Dungeon |
| **Rewards** | Request 100% → Unique Item/Skill (One-time) |
| **Visual Result** | Static 3D Village (Walkable) |
| **Agency** | Layout Optimization (Multiple Valid Solutions) |

### Limitations
- **No Crafting** — Pieces Pre-assembled
- **No Economy** — No Cost/Resource Management
- **No Simulation** — Village Static After Placement
- **No Persistence** — Next Village = Clean Slate
- **Single Era** — No Time Dimension
- **Toan Mandatory** — Atla Collection Forces Main Char

## DC2: Georama 2.0 — Causal Time Travel

### Pipeline
```
Dungeon → Geostones → Carpenterion → Blueprints → Materials + Polyn → Build
                                                                     │
                                                          Conditions (10/Area) → FUTURE ALTERED
```

### Mechanics
| Aspect | Implementation |
|--------|----------------|
| **Input** | Geostones (Any Char) → Blueprints → Materials (Buy/Find/Invent) |
| **Construction** | Component Assembly (Blueprint + Materials = Piece) |
| **Economy** | Polyn (Money) + Culture Points (CP) per Piece |
| **Constraints** | 10 Conditions/Area (Pop, CP, Specifics, NPC Placement, Terrain) |
| **Time Travel** | Past Build → Future Altered (Dungeons, Chests, Shops, Enemies) |
| **Progression Gate** | Conditions Met → Future Chests (Guaranteed HP/Def/Gourd/Pocket) |
| **Rewards** | Future Chests = Permanent Stats; Medals; Blueprints |
| **Visual Result** | Past/Future Pair (Visible Diff) |
| **Agency** | Multiple Valid Solutions per Condition; Material Choice Matters |

### Innovations over DC1
| DC1 | DC2 | Improvement |
|-----|-----|-------------|
| Place Pieces | **Build from Components** | Crafting Depth |
| No Cost | **Polyn + CP Economy** | Resource Management |
| Static Village | **Past→Future Causality** | Narrative + Gameplay Unity |
| Request % | **10 Explicit Conditions** | Clear Goals |
| Random Stat Items | **Guaranteed Future Chests** | Fair Mastery Rewards |
| Toan Only | **Any Char Collects Geostones** | Party Freedom |
| Single Era | **Past/Future Pair** | Time Travel Mechanic |

## Successor: Georama 3.0 — Persistent Multi-Era Simulation

### Core Concept
```
DESIGN (Parametric Blueprints)
    │
    ▼
BUILD (Materials + Energy + Time)
    │
    ▼
SIMULATE (Agents + Economy + Ecology + Threats)
    │
    ▼
PERSIST (World State Saved Across Sessions)
    │
    ▼
RIPPLE (Changes Propagate Across 4 Eras)
    │
    ▼
SHARE (Blueprint Library + Community Canon)
```

### Four Eras (Simultaneous Layers)
| Era | Time | Player Access | Unique Mechanics |
|-----|------|---------------|------------------|
| **Ancient Past** | -10,000 Years | Late Game | Ruins → Foundations; Lost Tech; Primal Threats |
| **Recent Past** | -100 Years | Early Game | Founding Families; Industrial Revolution; Steam/Clockwork |
| **Present** | Now | Always | Player Home Base; Active Economy; Live Events |
| **Future** | +100 Years | Mid Game | Consequences Visible; Tech Evolution; Cosmic Threats |

### Parametric Blueprints (Not Fixed Pieces)
```
Blueprint Definition:
   │
   ├─ SLOTS: Foundation / Frame / Walls / Roof / Interior / Exterior / Utility
   ├─ PARAMETERS: Size (X/Y/Z), Style (Cultural), Materials, Function
   ├─ CONSTRAINTS: Terrain, Budget, Era, Tech Level, Adjacency
   ├─ OUTPUTS: CP, Population, Resources, Defense, Research, Happiness
   └─ UPGRADE PATH: Tier 1 → 2 → 3 (Visual + Functional)
```

### Simulation Engine (Runs Continuously)
```
AGENTS (NPCs):
   ├─ NEEDS: Food, Shelter, Work, Social, Safety, Purpose
   ├─ SKILLS: Farming, Crafting, Combat, Research, Trade, Leadership
   ├─ SCHEDULES: Daily/Seasonal Routines
   └─ RELATIONSHIPS: Family, Friends, Rivals, Factions

ECONOMY:
   ├─ RESOURCES: Generated by Buildings + NPC Work
   ├─ MARKETS: Supply/Demand Pricing; Trade Routes
   ├─ TAXES: Player Sets Rates → Revenue + Happiness Impact
   └─ INVESTMENT: Fund Research/Infrastructure → Future Returns

ECOLOGY:
   ├─ FLORA/FAUNA: Populations Simulated; Migration; Extinction Risk
   ├─ WATER: Flow, Pollution, Irrigation, Flood/Drought
   ├─ CLIMATE: Seasons, Weather Events, Long-Term Shifts
   └─ CORRUPTION: Dark Energy Spreads; Player Contains/Exploits

THREATS:
   ├─ RAIDERS: Attack Weak Districts; Scale with Wealth
   ├─ MONSTERS: Spawn from Corruption; Evolve
   ├─ DISASTERS: Quakes, Floods, Fires, Blights
   └─ COSMIC: Meteor, Rift, Entity (Late Game)
```

### Ripple System (Causal Propagation)
```
PAST ACTION → PRESENT STATE → FUTURE OUTCOME
     │              │              │
     ▼              ▼              ▼
Ancient:    Recent:      Present:      Future:
Place       Build       Manage        See Results
Foundation  Factory     Economy       Automation
           ↓           ↓             ↓
       Recent Past   Present       Future:
       Pollution     Smog          Clean Tech
       (Choice)      (Consequence) (Redemption)
```

### Blueprint Library & Community
```
PERSONAL LIBRARY:
   ├─ Save Blueprints (With Simulation History)
   ├─ Tag: Era, Style, Function, Efficiency
   └─ Version Control (Iterate on Designs)

COMMUNITY LIBRARY:
   ├─ Publish Blueprints (CC-BY-SA)
   ├─ Fork/Remix Others' Designs
   ├─ Vote: "Canon" Designs Become NPC-Built in World
   ├─ Challenges: "Design a Self-Sustaining Village < 5000 CP"
   └─ Seasons: Themed Contests (Festival, Defense, Aesthetic)

CANON INTEGRATION:
   ├─ Top Voted → NPCs Build in Live World
   ├─ Player Visits → Sees Community Creations
   └─ Legacy: Your Designs Persist Across Playthroughs
```

## Design Principles for Future

1. **Construction = Gameplay, Not Menu** — Build in 3D; See Agents Use It
2. **Simulation = Consequence** — Every Choice Ripples; No "Correct" Answer
3. **Persistence = Legacy** — Your Cities Outlive Your Playthrough
4. **Community = Canon** — Best Designs Become World History
5. **Eras = Perspective** — Same Space, Different Time = Different Gameplay
5. **Crisis = Opportunity** — Disasters Reveal System Depth; Recovery = Mastery
