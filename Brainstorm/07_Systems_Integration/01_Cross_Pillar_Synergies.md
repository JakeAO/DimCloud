## Design Principle: "No System Is an Island"

> **Rule**: Every added system must connect to ≥3 existing systems.  
> **Rule**: No system exists in isolation — even cosmetics feed economy/reputation.

## System Interlock Rules (Enforced by Architecture)

### 1. Loot Tables = Simulation Outputs
```json
{
  "seed": "INFERNO_SPIRAL",
  "lootTables": {
    "common": ["Fire_Essence", "Iron_Component"],
    "rare": ["Fire_Crystal", "Inferno_Blueprint_Fragment"],
    "legendary": ["Inferno_Core", "Legacy_Fragment"]
  },
  "conditions": {
    "worldState.corruption > 0.7": { "add": ["Corrupted_Essence"] },
    "era == FUTURE": { "multiply": { "Essence": 1.5 } }
  }
}
```

### 2. Blueprint Requirements = Simulation State
```json
{
  "blueprint": "Hydro_Forge",
  "requirements": {
    "materials": { "Water_Crystal": 50, "Iron": 100 },
    "simulation": {
      "waterFlow > 100": true,
      "riverBiome == ACTIVE": true,
      "corruption < 0.3": true
    }
  },
  "outputs": {
    "essence": "Water",
    "unlocks": ["Aqueduct", "Irrigation_System"],
    "simulationEffects": {
      "irrigationEfficiency": 1.5,
      "farmYield": 1.3
    }
  }
}
```

## Emergent Gameplay Examples

### Scenario 1: "The Pollution Crisis"
```
Player builds Factory in Recent Past  →  Pollution Rises  →
  → Present: Smog Crisis  →  Player chooses Clean Tech  →
    → Future: Clean Energy Metropolis  →  New Blueprint: "Solar Forge"
      → Dungeon Seeds now drop "Solar Essence"
        → Weapon infusions: "Solar Blade" (Holy+Fire)
          → Mastery Track: "Clean Tech" unlocks "Solar Frame" Mech
```

### Scenario 2: "The Diplomat's Path"
```
Player uses Wolf Transform  →  Befriends Wolf Pack  →
  → Pack Leader joins as NPC  →  Unlocks "Pack Tactics" Skill Tree
    → Mech gains "Pack Interface" (AI-controlled wolf drones)
      → Dungeon Seeds: "Pack Hunter" objectives (Co-op with wolves)
        → Mastery: "Beast Master" → Unlocks "Alpha Frame" Mech
          → Georama: Wolf Den blueprint → Wolf population simulation
            → Ecology: Wolf population controls deer → vegetation changes
```

### Scenario 3: "The Community Canon"
```
Community designs "Sky Harbor" blueprint  →  Votes top 1%  →
  → NPCs build in Live World  →  All players see it  →
    → Future Era: Sky Harbor becomes Trade Hub  →
      → Economy: New trade routes  →  Prices shift  →
        → Dungeon Seeds: "Sky Trade Route" objectives  →
          → New Blueprint: "Airship Frame" (Community-designed!)
            → Legacy: Your design persists across all playthroughs
```

## Anti-Patterns to Avoid

| Anti-Pattern | Why It Fails | Solution |
|--------------|--------------|----------|
| **Island System** | No cross-feeds; dead content | Enforce ≥3 connections per system |
| **RNG Gatekeeping** | Mastery feels like luck | Deterministic Mastery Rewards |
| **Single-Track Progression** | Burnout; no choice | 7 Parallel Mastery Tracks |
| **Static World** | No replay value | Persistent Simulation + Community Canon |
| **Isolated Minigames** | Feels tacked on | Every Minigame Feeds Core (Essence/Blueprint/Rep) |
| **Single-Player Only** | No longevity | Async Community Layer (Blueprint/Seed/Canon) |
