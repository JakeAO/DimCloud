# Dark Cloud 2 — Medal, Spheda & Fishing Systems

## Medal System (Per-Floor Objectives)

### Overview
**396 Medals** across **127 Dungeon Floors**. Each floor offers up to 4 medals. Exchange at **Mayor Need** (Recruited Ch. 4) for rare rewards.

---

## Medal Types

| Medal | Requirement | Difficulty | Reward Points |
|-------|-------------|------------|---------------|
| **Time Trial** | Clear floor < Target Time | High (early) → Low (late) | 1 |
| **Wipeout** | Defeat ALL enemies (variant per chapter) | Medium | 1 |
| **Spheda** | Repair Time Distortion on floor | Low (with save states) | 1 |
| **Fishing** | Catch fish ≥ Target Length | RNG-heavy | 1 |
| **No Heal** | Clear without healing items | High | 1 |
| **Weapon Specific** | Clear using only [Weapon Type] | Varies | 1 |

> **Max 4 per floor** — Typically: Time Trial + Wipeout + (Spheda OR Fishing) + (No Heal OR Weapon Specific)

## Time Trial Strategies

### General Tips
- **Use Ridepod** with **Propeller Legs** (Ch. 3) or **Jet Hover** (Ch. 5)
- **Donny (Support)** — Unlocks all doors/chests → No key hunting
- **Skip ABS/Gold** — Let orbs despawn; saves 1–3 sec/floor
- **Memorize Layouts** — Seed fixed per entry; save state + retry
- **Weapon Choice** — One-shot weapon (Max's Gun / Monica's Sword)

### Chapter-Specific
| Chapter | Best Setup | Notes |
|---------|------------|-------|
| **Ch. 1 (Underground)** | Max + Strong Gun | Hardest; tight times; mimic RNG |
| **Ch. 2 (RB Wood)** | Ridepod Propeller + Donny | Pau shows map; easier |
| **Ch. 3 (Starlight)** | Ridepod Propeller | Spheda floors = dual objective |
| **Ch. 4 (Ocean's Roar)** | Ridepod Jet Hover | Water floors; hover ignores |
| **Ch. 5 (Gundor)** | Ridepod Jet Hover | Fire Squall = damage tick; hurry |
| **Ch. 7 (Moon Flower)** | Max Gun (Piercing) | Long floors; linear |
| **Ch. 8 (Zelmite)** | Max Ultimate Gun | Generous times; farm medals |

## Spheda (Golf) System

### Mechanics
- **Trigger**: Clear all enemies → "Spheda?" prompt (if distortion present)
- **Sphere Color** — Must be **OPPOSITE** of Distortion (Red↔Blue)
- **Color Change** — Sphere flips color on **each collision** (wall, object, ground)
- **Shots Limited** — Typically 3–5 shots per floor
- **Map Essential** — Full-screen map (Select → Large) shows distortion location

### Controls
| Input | Action |
|-------|--------|
| **L-Stick** | Aim direction |
| **× (Tap)** | Start Power Gauge |
| **× (Tap)** | Stop Power (Power = Distance) |
| **× (Tap)** | Hit Sphere (Timing = Accuracy) |
| **L1/R1** | Adjust Spin (Back/Top/Side) |
| **□** | Reset Sphere to Start |

### Advanced Techniques
| Technique | Use Case |
|-----------|----------|
| **Bank Shot** | Wall → Distortion (angle calculation) |
| **Object Chain** | Sphere → Tree → Rock → Distortion (color management) |
| **Power Gauge Memory** | Same distance = same power; memorize per floor |
| **Save State Per Shot** | Emulator: Save after each hit; reload on miss |

### Spheda Rewards (Mayor Need)

## Fishing & Finny Frenzy

### Fishing Basics
- **Rods**: Bait Rod (Ch. 1) → Lure Rod (Invent Ch. 2) → Ultimate Rods (Late)
- **Bait**: Evy (Universal), Insect, Minnow, Specific per fish
- **Mechanic**: Cast → Wait → "!" → × (Hook) → Mash × / Hold ← (Reel)
- **Fish HP** — Depletes while reeling; line tension (red = break risk)

### Fish Locations (Key)
| Area | Notable Fish | Min Length (Medal) |
|------|--------------|-------------------|
| Palm Brinks | Carp, Catfish | — |

### Finny Frenzy (Fish Racing)

#### Unlock
- **Invent Aquarium** (Photos: Fountain, Wooden Box, Window)
- **Ch. 4** — Talk to Firbit Girl in Town Square → Recruit Olivie

#### Aquarium Types
| Tank | Function | Capacity |
|------|----------|----------|
| **Rec Tank** | Feed fish; Enter Frenzy | 10 fish |
| **Breeding Tank** | 1 Male + 1 Female → Offspring | 1 pair |
| **Battle Tank** | 4 Fish fight → Stat gains (Risk: Death) | 4 fish |

#### Finny Frenzy Races
- **3 Heats** per entry (1st → 2nd → 3rd place required)
- **2 Laps** each; Auto-simulated
- **Fish Stats** determine performance:
  - **Speed** — Base velocity
  - **Stamina** — Maintains speed
  - **Boost** — Burst frequency
  - **Tenacity** — Recovers from bumps
  - **Endurance** — Survives collisions
  - **Battle** — Gains from Battle Tank

#### Frenzy Rewards
| Win | Reward |
|-----|--------|
| **Any Race** | Lure (Olivie sells all) |
| **Junior Cup** | Aquarium Upgrade |
| **Senior Cup** | Rare Fish Egg |
| **Master Cup** | **Ultimate Rod Blueprint** |

## Medal Completion Strategy

### Efficient Order
1. **Story First** — Unlock Donny (Ch. 2), Pau, Gerald, Milane, Adel
2. **Ch. 1–3 Medals** — Do during story (easier with Ridepod Propeller)
3. **Ch. 4–5 Medals** — Jet Hover makes trivial
4. **Ch. 7 Medals** — Linear; good Gun clears fast
5. **Ch. 8 (Zelmite)** — Farm remaining; best EXP/Gold too
6. **Spheda/Fishing** — Save states; do in batches

### Save-State Workflow (Emulator)
```
Save State at Floor Start
    │
    ├─▶ Time Trial → Save if Gold
    ├─▶ Wipeout → Save if Gold
    ├─▶ Spheda → Save State PER SHOT → Reload on miss
    └─▶ Fishing → Save State PER CAST → Reload on short fish
```
